## Jetbrains Yang Plugin

[![GitHub tag](https://img.shields.io/github/tag/snlab-freedom/Jetbrains-YANG.svg)](https://github.com/snlab-freedom/Jetbrains-YANG/tags)
[![GitHub release](https://img.shields.io/github/release/snlab-freedom/Jetbrains-YANG.svg)](https://github.com/snlab-freedom/Jetbrains-YANG/releases)
[![GitHub issues](https://img.shields.io/github/issues/snlab-freedom/Jetbrains-YANG.svg)](https://github.com/snlab-freedom/Jetbrains-YANG/issues)

It's a Jetbrains IntelliJ plugin for [YANG modeling language](https://github.com/YangModels/yang).

It will provide the following features in the first formal release:

- [x] Syntax highlighting
- [ ] Code completion
- [ ] Refactoring
- [ ] Code formating

Features below are in the future plan:

- [ ] Documentation lookup
- [ ] Module dependency graph generator
